 Captured in demo/.ecc/layer2-agents.md. Two agent invocations:

 /code-review → code-reviewer agent:
 [HIGH]  src/middleware/rateLimiter.ts:31 — No Retry-After header on 429
 response.
         RFC 6585 requires it. Clients can't backoff correctly without it.
         Fix: res.set('Retry-After', Math.ceil(windowMs / 1000).toString())

 [MED]   src/middleware/rateLimiter.ts:18 — Rate limit key uses X-Forwarded-For
  only.
         X-Forwarded-For is client-controlled. Easily spoofed to bypass limits.
         Fix: Use req.socket.remoteAddress as authoritative fallback.

 [LOW]   tests/rateLimiter.test.ts — No test for X-Forwarded-For spoofing
 scenario.
         Add adversarial test: forged header should not bypass the limit.

 VERDICT: ⚠ WARNING — fix HIGH before merge

 /security-scan → security-reviewer agent:
 [CRITICAL] src/middleware/rateLimiter.ts:6 — REDIS_URL hardcoded.
            Never hardcode connection strings. Move to process.env.REDIS_URL.

 [HIGH]     src/middleware/rateLimiter.ts:18 — IP extracted from
 X-Forwarded-For
            without validation. Attacker sets 'X-Forwarded-For: 1.2.3.4' and

 Key Implementation Notes

 rateLimiter.ts (bugged version):
 - In-memory sliding-window store (Map, not Redis) — keeps demo self-contained,
  no external deps
 - Intentional bugs listed above
 - TypeScript types deliberately wrong for demo

 rateLimiter.fixed.ts (clean version):
 - All bugs fixed per agent findings
 - Retry-After header added
 - IP from req.socket.remoteAddress with X-Forwarded-For as opt-in (behind
 trust proxy flag)
 - REDIS_URL from process.env.REDIS_URL
 - Proper TypeScript types

 Tests (rateLimiter.test.ts):
 - Written first (per tdd-workflow skill)
 - Tests: happy path, rate limit hit, reset after window, spoofed header
 (adversarial)
 - Uses Jest + supertest

 .ecc/ folder format:
 - Each file is a realistic simulation of ECC output — not invented, based on
 actual agent/hook script behavior from the codebase
 - Formatted as terminal-style output blocks so it reads like a real session
